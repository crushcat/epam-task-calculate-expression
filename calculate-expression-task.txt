Условие.
Посчитать результат арифметического выражения. Арифм. выражение состоит из цифр 0-9, арифм. операций "+" (плюс), "-" (минус), "x" (умножить), ":" (разделить) и круглых скобок "(", ")". Скобки имеют наивысший приоритет (1), операции умножить и разделить (2), затем остальные ариф. операции (3).

Решение должно включать проверку на ошибки, верно введенные входные параметры.
Ошибки должны выводится в тег <div/> для сообщений, желтым цветом на красном фоне (CSS использовать нельзя).

Так же должна быть кнопка, которая проверяет и выводит сообщение белым цветом на зеленом фоне, является ли введенная строка математическим выражением (button Is valid expression).

Данные вводятся, в HTML (<input/> или <textarea/>). Для расчетов должна быть кнопка "Calculate" (<button/>).
Результат должен выводится в тег <div/> для сообщений результат вычислений должен выводится белым цветом на синем фоне.

ПРИМЕЧАНИЕ:
eval - использовать нельзя 

Тест 1.
Входные данные: 5 x 2 - 3
Результат: 7

Тест 2.
Входные данные: 5 x (2 - 3)
Результат: -5

Тест 3.
Входные данные: 2 + (7 x (6 - 8 : 4) + 2 x (9 + 49 : 7)) + 5 x (2 - 3 x 7)
Результат: -33

Тест 3.
Входные данные: 7 x (12 : 3 + 2 x 4) / (3 x 3 + 5)
Результат: 6
